# Fuego-Bot

## Description

Community run discord bot for learning purposes only.

## Installation

```bash
$ npm install
```

copy .example.env and rename to .env
input required tokens

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
